# Animation.auto 

Faceless video & anime automation platform

## Quick Deploy to Render
1. Fork this repo
2. Connect to Render.com
3. Add MongoDB Atlas & Razorpay keys to Environment Variables
4. Deploy!

## Features
Unlimited free for owner (phone: 9999999999)  
UPI payments (Razorpay)  
Faceless video generator  
Script-to-anime converter  
Real-time progress  
Dashboard & analytics
